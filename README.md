# ReactPokédex
A web application which lists pokémon data from [PokeAPI.co](https://pokeapi.co/api/v2/). Made using React and material-ui.

## To run
- Clone into repo
- Run `npm i`
- Run `npm start`
- Visit `localhost:3000`
